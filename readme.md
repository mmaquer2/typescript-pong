# Pong with Excalibur.js in TypeScript

![Game Photo](./images/Screenshot_pong3.png)

## Setup and Installation

```bash
# Install dependencies
npm install

# Run the game
npm run start

# go to localhost:1234 in your browser of choice
# Play the Game!

```
